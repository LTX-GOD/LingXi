# Contributing

感谢你为 LingXi 公开版做贡献。

## 提交前请确认

- 改动只涉及公开边界内的代码、测试或文档。
- 没有提交密钥、令牌、日志、数据库、抓取归档或运行产物。
- 没有并入官方工具包、私有扩展、私有 PoC 或 Sliver 资产。
- 新增依赖与 vendored 内容的许可证与 `GPL-3.0-only` 兼容。
- 至少运行与改动相关的最小验证命令。

## 建议流程

1. 先讨论设计或问题背景。
2. 保持改动聚焦，优先做公开版核心路径上的修复或增强。
3. 为可选扩展保留“缺省关闭、缺失不崩”的行为。
4. 更新 README / QUICKSTART / 测试，保持文档与代码一致。

## 不接受的内容

- 真实比赛环境配置
- 官方或私有扩展资产
- 私有知识库快照、数据库、日志、抓取结果
- 许可证来源不清晰的复制代码
