# Security Policy

## 支持范围

本仓库只处理公开版 LingXi 的安全问题。

不在公开仓库范围内的内容包括：

- 私有论坛扩展
- 私有 PoC 包
- Sliver client / operator 配置
- 任何比赛环境专用资产

## 报告内容

欢迎报告以下问题：

- 敏感信息泄露
- 公开版默认配置导致的危险行为
- 许可证/第三方代码边界问题
- 公开版核心路径中的安全缺陷

## 提交前请先自查

- 不要在 issue 中粘贴真实密钥、令牌、数据库、日志或比赛环境 URL。
- 如果问题与私有扩展有关，请先剥离私有实现细节，只描述公开接口与现象。

## 修复原则

- 缺省关闭高风险扩展
- 缺失私有扩展时应优雅降级
- 安全修复优先同步 README 与配置模板
